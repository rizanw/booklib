# 📦 Backend - Book Library CRUD App

## 🛠️ Setup

### 1. Prerequisites

- Go ≥ 1.24
- postgresql (you can use docker-compose here)

### 2. Environment Variables

Update `config.yaml` file inside `files/etc/booklib`

### 3. Run Backend Server

```bash
go mod tidy
go run cmd/http/main.go
```

or using make:

```bash
make run
```
